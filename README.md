# stylelint-order

## 说明

基于stylelint-order的样式排序插件，用于HBuilder X编译器中的css、less、sass样式排序和修正。

## 配置文件，如何配置选项？

stylelint-order配置文件是.stylelintrc.js，如果项目中存在.stylelintrc.js文件，以项目中的配置为准。
点击菜单工具 -> 插件配置 -> stylelint-order -> .stylelintrc.js，即可打开配置文件。
